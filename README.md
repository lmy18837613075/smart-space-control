# 智能空间控制系统

基于 ESP32 的智能空间控制系统 — 物联网学习项目

## 项目简介

这是一个**物联网全栈学习项目**，通过实际开发一个智能空间控制系统，学习物联网核心技术：

- 嵌入式开发（ESP32 + 传感器）
- 物联网通信协议（MQTT）
- 后端开发（FastAPI）
- 前端开发（Web 管理界面）
- 云边协同架构
- AI 智能决策

通过 ESP32 传感器采集环境数据，结合 MQTT 通信和 AI 算法，实现智能温控、红外遥控、设备联动等功能。

## 核心特性

- **三层决策机制**：本地门控 → 云端精细分析 → 本地兜底
- **CoolIndex™ 凉度指数**：综合温度/湿度/风速/辐射计算舒适度
- **三种空间场景**：小型(宿舍)/中型(办公室)/大型(教室)
- **云端降级策略**：本地优先，云端增强，断网也能用
- **第三方设备接入**：小米(m python-miio)、美的 API 等

## 技术架构

```
┌─────────────────────────────────────┐
│  用户交互层 (Web/App)               │

└─────────────────────────────────────┘
```

## 技术栈

| 层级 | 技术 |
|------|------|
| 固件 | ESP32 + Arduino C++ |
| 通信 | MQTT |
| 后端 | Python + FastAPI |
| 数据库 | SQLite/PostgreSQL |
| 前端 | Vue3 + Element Plus |
| 部署 | Docker Compose |

## 学习路线

这个项目按照物联网技术栈自底向上学习：

```
1. 感知层 → ESP32 固件开发、传感器驱动
2. 通信层 → MQTT 协议、网络编程
3. 平台层 → FastAPI 后端、数据库设计
4. 应用层 → Web 前端、用户交互
5. 智能层 → AI 算法、边缘计算
```

每个阶段都有明确的学习目标和实践任务，适合系统性学习物联网开发。
